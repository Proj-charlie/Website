Up-keeping the performance and stability of a Minecraft server requires great effort from administrators. Making sure that add-ons work whilst simultaneously chasing after bugs and inconsistencies. This all takes time from what server administrators really should be doing, growing their server and focussing on the existing community.

With every upcoming Minecraft update, performance starts to degrade on a production server. This allows for an almost continuous cycle of chasing performance and stability issues.

In our own adventures, we faced the same problems. After spending countless hours working on patches, we quickly realized the importance of delegation. If we spent less time on chores, we would be able to refocus that time to growing the server and giving our players what they want: new features, events, and game modes. We are ready to share our knowledge with you.
