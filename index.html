    Multiplayer servers

Up-keeping the performance and stability of a Minecraft server requires great effort from administrators. Making sure that add-ons work whilst simultaneously chasing after bugs and inconsistencies. This all takes time from what server administrators really should be doing, growing their server and focussing on the existing community.

With every upcoming Minecraft update, performance starts to degrade on a production server. This allows for an almost continious cycle of chasing performance issues and other matters on the subject of stability.

In our own adventures, we faced the same problems. After spending countless hours working on patches we quickly realised the importance of delegation. If only we could spend less time on these chores, we'd be able to grow the server exponentially and give our players what they want. More features, events or gamemodes.
